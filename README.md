# Пример кода рендеринга с адаптивной версткой
### Задача
Данный репозиторий содержит решение тестового задания. Задача поставлена следующим образом:

Есть ответ от API для «Instagram», надо вывести результат выполнения запроса по предоставленному макету. При нажатии на сердце, должен быть отображен «alert» с идентификатором соответствующей записи. Код должен быть написан максимально грамотно, как будто этот код планируется использовать и расширять. Страница должна получиться адаптивной (в макете изображено два состояния). Задание выполняется без использования клиентских библиотек.

### Использование

Работоспособность примера может быть проверена следующим образом:
- скачайте архив с репозиторием
- запустите файл index.html
